# Ola Dashboard Analytics

##  Description
Developed an interactive dashboard to analyze ride booking data and extract insights on customer demand and revenue trends. Cleaned and processed data using Pandas, performed exploratory data analysis to identify peak hours and high-demand locations, and built visualizations to support data-driven pricing and driver allocation decisions.

## Tech Stack
- Python (Pandas, NumPy)
- Data Visualization (Power BI / Tableau / Matplotlib)

## Key Work
- Data cleaning and preprocessing  
- Exploratory Data Analysis (EDA)  
- Dashboard creation and visualization  
- Insight generation for business decisions  

##  Outcome
Enabled identification of demand patterns and revenue trends to improve operational efficiency.
