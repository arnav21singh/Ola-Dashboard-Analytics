# 🚖 Ola Dashboard Analytics

## 📌 Description
Developed an interactive dashboard to analyze ride booking data and extract insights on customer demand and revenue trends. Performed data cleaning and exploratory data analysis using Python (Pandas) to identify peak hours and high-demand locations, enabling data-driven decision-making.

## ⚙️ Tech Stack
- Python (Pandas, NumPy)
- SQL
- Power BI

## 📊 Key Analysis Performed

### 🔹 SQL Analysis
- Retrieved successful bookings  
- Calculated average ride distance per vehicle type  
- Identified top customers by ride count  
- Analyzed cancellations by customers and drivers  
- Computed total revenue from successful bookings  

### 🔹 Power BI Dashboard Insights
- Ride volume trends over time  
- Booking status distribution  
- Revenue by payment method  
- Top customers by booking value  
- Ride distance distribution  
- Customer and driver rating analysis  

## 🚀 Outcome
Identified key patterns in demand, revenue, and customer behavior, helping optimize pricing strategies and operational efficiency.
